## ガイドの使い方

### 初めてDLTパイプラインを実装する場合

1. **[DLTパイプライン完全ガイド](./DLT_COMPLETE_GUIDE.md)** を参照
2. 必要に応じて、詳細ガイドを参照

### 特定の作業を行う場合

- **DLTパイプラインを作成する**: [DLTパイプライン完全ガイド](./DLT_COMPLETE_GUIDE.md) の「ステップ1: 統合DLTパイプラインの作成」セクション
- **データ整合性を確認する**: [DLTパイプライン完全ガイド](./DLT_COMPLETE_GUIDE.md) の「データ整合性確認」セクション
- **トラブルシューティング**: [DLTパイプライン完全ガイド](./DLT_COMPLETE_GUIDE.md) の「トラブルシューティング」セクション

## ガイドの整理について

以前は、DLT関連のガイドが複数に分散していてわかりにくい状態でした。現在は、以下のように整理されています：

- **統合ガイド**: [DLTパイプライン完全ガイド](./DLT_COMPLETE_GUIDE.md) に全ての情報を統合
- **実装者向けガイド**: [DLT実装ガイド](./DLT_IMPLEMENTATION_GUIDE.md) で実装レベルの詳細を提供
- **不要なガイド**: 統合ガイドに統合されたため削除済み
